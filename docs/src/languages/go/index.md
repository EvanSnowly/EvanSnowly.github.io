## Goroutine 
Goroutine 是 Go 语言中用于并发执行的轻量级线程。与传统的线程相比，goroutine 的创建和销毁成本很低，可以在同一个程序中轻松创建多个 goroutine 而不会导致性能下降。Goroutines 之间通过通道（channel）进行通信，实现了并发编程的简洁和安全。
