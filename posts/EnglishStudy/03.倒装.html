                  # Only in this way can we improve English

                  # 倒装的作用：表强调 但不是每个句子都可以倒装，必须在满足定条件下，才能进行相应的倒装。

                  # 一，完全倒装：关键词谓语全部+主语+其他

                  # 1.表方位的副词位于句首，句子全倒装 there,here,away,out/outside,in,up,down

                  # Out rushed the little child

                  # 2.表方位的介词短语位于句首，句子全倒装 in/on/at/over/under/across...+the n.

                  # At the top of the mountain stands a temple.

                  # 注意：当主语是人称代词时(you/I/we/he/she/it/hey 即使有方位副词/介词短语位于句首，也不全倒装

                  # 二，部分倒装：关键词+be情态/助动词+主语+谓语（另一半)+其他

                  # 1.否定副词或短语位于句首时，句子半倒装

                  # 否定词:not,never,nowhere,seldom,little,few,rarely, hardly,scarcely,no sooner,neither,nor

                  # 否定短语:in no case,in no sense,by no means,.in no way,under no circumstances,at no time,on no occasion, on no account(绝不)

                  # Never will I give up to difficulties.

                  1. # "一...就...”：hardly/scarcely...when ,no sooner...than

                  2. # Hardly/Scarcely/No sooner位于句首，主句半倒。 主句用had done,从句用did(主倒从不倒)

                  3. # Hardly had I gone home when it started to rain heavily.

                  4. # not...until“直到.…才”Not until位于句首，主句半倒（且时态一般为过去)

                  5. # not only,but(also) “不但.…而且？ Not only位于句首，半倒（前倒后不倒）

                  # 2.so/such..that“如此.以致于 So/Such+所修饰部分位于句首，主句半倒

                  # So difficult was the exam that he failed to pass it

                  # 3.Only+状语位于句首，主句半倒

                  1. Only in this way
                  2. can we
                  3. Only if...半倒
                  4. Only when...半倒

                  # 4.虚拟倒装

                  # 在if引导的非真实条件句中，可以把were/had(助 动词)/should提到主句之前，同时省略if。

                  # Were I you,I would continue to study

                  # 三,特殊倒装

                  # 1.表“也”和“的确

                  # 表肯定“也.”：so+be/情态/助动词+主语

                  # 表否定“也不.”：neither/nor+be/情态/助动词+主 语

                  # 当前面既有肯定又有否定或有两个谓语动词时，用 so it is with sb

                  # 表“的确”：so+主语+be/情态/助动词（只能表肯定)

                  # 2.as表“虽然"必须用倒装语序：

                  # adj./adv./n.+as+主语+谓语

                  # Rich as he is,I don't envy him.
