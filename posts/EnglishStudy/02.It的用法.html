                    it的用法

                    wuxuefeng大约 2 分钟学习English
                    此页内容
                    • 一,it作代词
                      • 1.it代指前文提到过的一切事物译为“它，这,那''
                      • 2.it代指人：指代性别不明的婴儿、身份性别不明确的人。
                      • 3.it代指代词：
                      • 4.it表示天气、气候、温度、时间、距离、价值等
                    • 二,it作形式主语
                      • 1.It is+adj.+(for/of)sb to do sth做.…是.的
                      • 2.It is+adj./done/n.+主语从句
                      • 3.It takes(sb.)+时间+to do sth.某人做花.…时间
                    • 三、it作形式宾语
                    • 四,it用于强调句

                    # 一,it作代词

                    # 1.it代指前文提到过的一切事物译为“它，这,那''

                    # I dropped my watch and it broke.

                    # 2.it代指人：指代性别不明的婴儿、身份性别不明确的人。

                    # Is it boy or a girl??

                    # There is a knock on the door must be the postman

                    # Someone is calling you.Who is it?

                    # 3.it代指代词：

                    # it代指his,that,something,anything,nothing等代词

                    # What's this?It's a new machine.

                    # Nothing is wrong,is it?

                    # 4.it表示天气、气候、温度、时间、距离、价值等

                    # It is raining.

                    # It is sunny today

                    # It is 3 pm now.

                    # 二,it作形式主语

                    # 1.It is+adj.+(for/of)sb to do sth做.…是.的

                    of用于指某人的性格、属性、特征
                    for表示对某人来说

                    # It is hard him to make up his mind.

                    # It was foolish her to say such a thing.

                    # 2.It is+adj./done/n.+主语从句

                    # It's unknown when he will come.

                    # When he will come is unknown.

                    # 3.It takes(sb.)+时间+to do sth.某人做花.…时间

                    # It takes years to master a new language.

                    # 三、it作形式宾语

                    # 当不定式、动名词、从句等复杂成分用作宾语且其后跟有宾语补足语时，通常会在宾语补足语前使用形式宾语，而将真正的宾语移至句末，称为t作形式宾语。

                    # sb.+(think/believe/consider/feel/find/make/take regard等)+it+adj.n.+to do sth./从句某人认为觉得做.…是.…的=

                    # I find it to do the job well.

                    # I find it important to take notes in the class

                    # I find it important to take notes in the class

                    # I fell it a great honor to be invited to take part in the party

                    # I take it for granted that you finish your work on time

                    # 四,it用于强调句

                    # It is/was+被强调部分+that/who+其他部分

                    # Is/Was+it+被强调部分+that/who+其他部分？

                    # It is not until+被强调部分+hat+其他部分

                    # 1.当被强调部分是作主语的人称代词，用主格 形式

                    # 2.当被强调部分是时间、地点，只能用that,不 能用when,where。

                    # 3.强调句判定方法：把it is that/who去掉，剩 余部分能组合成完整的句子。
